Detailed Description

template<class Type>
class Foam::cellDof< Type >

Degree-of-freedom container for a DG cell.

Template Parameters
TypeField value type (e.g. scalar, vector, tensor)

cellDof stores all degrees of freedom (DoF) associated with a single DG cell for a given field type.

The DoFs typically correspond to coefficients of polynomial basis functions defined on the reference cell and mapped to physical space.

This class is intentionally lightweight and contains no geometric information; it is purely a data container used by higher-level DG field and solver classes.
