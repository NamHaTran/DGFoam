Detailed Description

template<class Type>
class Foam::cellGaussField< Type >

Field values at Gauss points of a DG cell.

Template Parameters
TypeField value type (e.g. scalar, vector, tensor)

cellGaussField stores field values evaluated at Gauss integration points inside a single DG cell.

The values are typically obtained by interpolating cell-local degrees of freedom (DoF) using basis functions provided by the DG geometric cell.

This class does not own geometric data or DoF storage; it holds non-owning pointers to dgGeomMesh, dgGeomCell, and cellDof<Type>.
