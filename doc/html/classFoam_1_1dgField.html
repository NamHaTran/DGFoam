Detailed Description

template<class Type>
class Foam::dgField< Type >

DG field storing DoFs and Gauss-point values.

Template Parameters
TypeField value type (e.g. scalar, vector, tensor)

dgField represents a field discretized using the Discontinuous Galerkin method over the entire mesh.

The class stores:

  • Degrees of freedom (DoF) per cell via dofField<Type>
  • Precomputed GaussField<Type> values for each cell

A dgField may own a DoF field (primary DG variables) or act as an auxiliary field without DoFs (derived quantities).

The class derives from regIOobject to integrate with OpenFOAM I/O and objectRegistry mechanisms.
