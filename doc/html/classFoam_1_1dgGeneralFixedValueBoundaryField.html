Detailed Description

template<class Type>
class Foam::dgGeneralFixedValueBoundaryField< Type >

Strong Dirichlet boundary condition for DG fields.

Enforces a prescribed boundary value using a ghost-state reflection.

The Dirichlet condition is imposed strongly by mirroring the interior (minus-side) state about the prescribed value:

\[ U^{+} = 2\,U_{\mathrm{bc}} - U^{-} \]

where:

  • \(U^{-}\) is the interior (minus-side) value
  • \(U^{+}\) is the ghost (plus-side) value
  • \(U_{\mathrm{bc}}\) is the prescribed boundary value

No explicit gradient modification is applied; the boundary value fully constrains the solution at the interface.
