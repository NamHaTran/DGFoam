Detailed Description

template<class Type>
class Foam::dgGeneralInletOutletBoundaryField< Type >

Inlet–outlet boundary condition for DG fields.

Selects inlet/outlet behaviour per face Gauss point based on the sign of \(\mathbf{U}\cdot\mathbf{n}\) evaluated at that Gauss point.

Inlet ( \(\mathbf{U}\cdot\mathbf{n}<0\)):

\[ \phi^{+} = 2\,\phi_{\mathrm{inlet}} - \phi^{-} \]

Outlet ( \(\mathbf{U}\cdot\mathbf{n}\ge 0\)):

\[ \phi^{+} = \phi^{-} \]

Requirements:

  • A velocity field named "U" of type dgField<vector> must be available.
  • The inlet value must be provided via dictionary entry inletValue.
