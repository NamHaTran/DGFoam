Detailed Description

template<class UType, class FType>
class Foam::dgGeneralPDETerm< UType, FType >

Generic DG PDE term for residual assembly.

Template Parameters
UTypeType of solution variable (scalar or vector)
FTypeType of flux (vector or tensor)

This class represents a single PDE operator contribution to the DG residual of one cell.

The residual is assembled in the form:

\[ R_i = - \int_{\Omega} \nabla \phi_i : \boldsymbol{F}\,\mathrm{d}V + \int_{\partial\Omega} \phi_i (\boldsymbol{F}^{\ast}\cdot\boldsymbol{n})\,\mathrm{d}S \]

Supported type combinations:

  • UType = scalar, FType = vector
  • UType = vector, FType = tensor

The numerical surface flux \(\boldsymbol{F}^{\ast}\) is computed through dgFluxSolverManager based on the PDE term name.

This class:

  • Operates on a single cell
  • Owns no global data
  • Is used repeatedly during residual assembly
