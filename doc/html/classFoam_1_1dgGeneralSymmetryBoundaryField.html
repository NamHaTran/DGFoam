Detailed Description

template<class Type>
class Foam::dgGeneralSymmetryBoundaryField< Type >

Symmetry boundary condition for DG fields.

Enforces a symmetry-plane constraint at face Gauss points by reflecting the interior (minus-side) state with respect to the boundary unit normal.

Scalar fields:

\[ \phi^{+} = \phi^{-} \]

Vector fields:

\[ \boldsymbol{\phi}^{+} = \boldsymbol{\phi}^{-} - 2(\boldsymbol{\phi}^{-}\cdot\mathbf{n})\,\mathbf{n} \]

Only interior values and geometric information are used. No dictionary parameters are required.
