Detailed Description

DG geometric mesh wrapper around fvMesh.

dgGeomMesh constructs and stores all DG-specific geometric entities derived from an fvMesh.

Responsibilities include:

  • Creating and storing dgGeomFace and dgGeomCell objects
  • Managing shared reference-face (dgRefFace) data
  • Managing reference-cell (dgRefCell) data for different cell types
  • Providing connectivity information required for DG surface and volume integration

The class acts as the central geometric data container for DG discretization.
