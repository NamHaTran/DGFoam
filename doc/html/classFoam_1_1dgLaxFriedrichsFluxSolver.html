Detailed Description

Local Lax–Friedrichs (Rusanov) numerical flux.

Implements the local Lax–Friedrichs (Rusanov) flux for DG formulations.

The numerical flux is defined as:

\[ \boldsymbol{h} = \frac{1}{2} \left( \boldsymbol{F}(\boldsymbol{U}^{-}) + \boldsymbol{F}(\boldsymbol{U}^{+}) \right)\cdot\boldsymbol{n} - \frac{1}{2} C \left( \boldsymbol{U}^{+} - \boldsymbol{U}^{-} \right) \]

where:

  • \(\boldsymbol{U}^{-}\) is the interior (minus-side) state
  • \(\boldsymbol{U}^{+}\) is the exterior (plus-side) state
  • \(\boldsymbol{F}\) is the physical flux
  • \(\boldsymbol{n}\) is the outward unit normal

The dissipation coefficient \(C\) is evaluated locally from flow velocity and acoustic speed. Optional Mach-based scaling can be enabled via dictionary settings.

This flux is robust but introduces stronger numerical dissipation than approximate Riemann solvers such as HLLC or Roe.
