Detailed Description

Reference-cell data for DG discretization.

dgRefCell stores all reference-space data required by the Discontinuous Galerkin method for a given cell type and polynomial order.

This includes:

  • Gauss quadrature points and weights in reference coordinates
  • Basis functions evaluated at Gauss points
  • Derivatives of basis functions with respect to reference coordinates \(\eta_1\), \(\eta_2\), and \(\eta_3\)

The data is generated once and shared among all physical cells of the same type and order to avoid redundant computations.
