Detailed Description

template<class Type>
class Foam::faceGaussField< Type >

Gauss-point field on faces of a DG cell.

Template Parameters
TypeField value type (e.g. scalar, vector, tensor)

faceGaussField stores field values evaluated at Gauss integration points on all faces of a DG cell.

For each Gauss point on a face, the class stores:

  • Minus-side value (owner cell)
  • Plus-side value (neighbour cell or boundary)

The class is designed for cell-centric DG formulations and supports flux computation across cell interfaces.

This class does not own geometry or DoF storage; it holds non-owning pointers to dgGeomMesh, dgGeomCell, dgGeomFace, and cellDof<Type>.
