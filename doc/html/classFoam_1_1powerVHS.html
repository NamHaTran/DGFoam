Detailed Description

Variable-Hard-Sphere (VHS) power-law viscosity model.

powerVHS implements the Variable-Hard-Sphere (VHS) model with a power-law temperature dependence of viscosity:

\[ \mu(T) = \mu_{\mathrm{ref}} \left(\frac{T}{T_{\mathrm{ref}}}\right)^{\omega} \]

The reference viscosity is computed from kinetic theory as:

\[ \mu_{\mathrm{ref}} = \frac{15 \sqrt{\pi m k_B T_{\mathrm{ref}}}} {2 \pi d_{\mathrm{ref}}^2 (5 - 2\omega)(7 - 2\omega)} \]

where:

  • \(m\) is the molecular mass
  • \(k_B\) is the Boltzmann constant
  • \(d_{\mathrm{ref}}\) is the reference molecular diameter
  • \(T_{\mathrm{ref}}\) is the reference temperature
  • \(\omega\) is the VHS power-law exponent

The Prandtl number is assumed constant:

\[ \mathrm{Pr}(T) = \mathrm{Pr}_0 \]

Thermal conductivity is computed in the base transportLaw as:

\[ \kappa(T) = \frac{\mu(T)\,C_p(T)}{\mathrm{Pr}(T)} \]

Notes
  • Molecular mass is read in \(\mathrm{g/mol}\) and converted internally to \(\mathrm{kg/molecule}\)
  • \(d_{\mathrm{ref}}\) is specified in \(\mathrm{m}\)
  • \(T_{\mathrm{ref}}\) is specified in \(\mathrm{K}\)
  • \(k_B\) is specified in \(\mathrm{J/K}\)
  • \(N_A\) is specified in \(\mathrm{mol^{-1}}\)
Example
transportCoeffs
{
type powerVHS;
// Required
molMass 28.0134; // [g/mol] N2
dRef 3.7e-10; // [m]
TRef 300; // [K]
omega 0.74; // [-]
Pr 0.72; // [-]
// Optional overrides
// kB 1.380649e-23; // [J/K]
// NA 6.02214076e23; // [1/mol]
}
