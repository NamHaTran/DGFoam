27Class
28 Foam::dgHLLEFluxSolver
29
30Description
31 HLLE (Harten–Lax–van Leer–Einfeldt) approximate Riemann solver.
32
33 This flux uses only the bounding wave speeds S_L and S_R to construct a
34 single intermediate state between the left (Minus/P-side) and right
35 (Plus/N-side) states. HLLE does not resolve the contact or shear waves,
36 and therefore produces a more diffusive but very robust and
37 positivity-preserving flux.
38
39 Key properties
40 - Robust for strong shocks and large discontinuities.
41 - Positivity-preserving for density and pressure under standard CFL
42 conditions.
43 - No use of a contact wave → more diffusive than HLLC.
44 - No star-state reconstruction → simpler and cheaper to evaluate.
45 - Suitable for any conserved quantity (rho, rhoE, rhoU, scalar).
46
47 Procedure (local 1D Riemann problem along face normal n):
48 1. Project left and right physical fluxes onto n.
49 2. Compute left/right normal velocities and sound speeds.
50 3. Estimate wave speeds S_L and S_R using either:
51 * Davis (S_L = min(Un^- - a^-, Un^+ - a^+),
52 S_R = max(Un^- + a^-, Un^+ + a^+)), or
53 * Roe–Einfeldt averaging for reduced numerical diffusion.
54 4. Construct the HLLE flux:
55 f_HLLE = f_L, if S_L >= 0
56 = f_R, if S_R <= 0
57 = ( S_R f_L - S_L f_R
58 + S_L S_R (U_R - U_L) ) / (S_R - S_L), otherwise
59 5. For scalar equations the Cartesian flux is f_HLLE * n.
60 For vector/momentum equations the Cartesian flux is f_HLLE as a vector
61 obtained from the tensor flux projection; no multiplication by n.
62
63 Notation
64 - Minus = owner cell (P-side)
65 - Plus = neighbour cell (N-side)
66 - n = unit normal pointing P → N
67 - U_L/R = conserved variables at Gauss point
68 - f_L/R = physical normal fluxes projected onto n
