27Class
28 Foam::dgFluxSolver
29
30Description
31 Base class for numerical flux solvers that operate on vector-valued
32 fluxes. The interface accepts Plus/Minus flux vectors (Cartesian)
33 and returns the numerical flux vector.
34
35 Notes
36 - New convention:
37 * Plus = neighbour cell (N-side)
38 * Minus = owner cell (P-side)
39 - This class only works with Foam::vector (no tensor support).
40 - A fieldsContext pointer may be set for field lookup/registration.
41
42 Flux solvers's category:
43

45 ├── exactRiemannFluxSolver/
46 │ ├── Godunov.C
47 ├── approximateRiemannFluxSolver/
