27Class
28 Foam::GaussField
29
30Description
31 Wrapper class for Gauss-point field data of a DG cell.
32
33 This class aggregates Gauss-point values inside a cell and on its faces.
34 It provides a unified interface to access and operate on:
35 - Cell interior Gauss-point values
36 - Face Gauss-point values (minus/plus sides)
