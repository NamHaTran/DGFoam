Detailed Description

Utility functions for DG boundary patch classification.

This namespace provides a lightweight set of helper functions to classify OpenFOAM boundary patches into logical categories used by the DG framework:

  • Interface patches (e.g. processor boundaries)
  • General physical boundary patches (walls, inlets, outlets, symmetry)
  • Constraint or special patches (e.g. overset)

The classification is based solely on the fvPatch type string and is intentionally kept simple and extensible.
