Detailed Description

Basis-function data on a reference face.

Stores basis-function values and their derivatives evaluated at Gauss quadrature points on a reference face.

The derivatives are expressed with respect to the reference coordinates \(\eta_1\), \(\eta_2\), and \(\eta_3\).
