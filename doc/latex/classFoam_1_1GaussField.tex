Combined Gauss-\/point field for a DG cell. 


\begin{DoxyTemplParams}{Template Parameters}

\hline
\end{DoxyTemplParams}
\doxylink{classFoam_1_1GaussField}{Gauss\+Field} acts as a high-\/level container that combines\+:
\begin{DoxyItemize}
\item cell\+Gauss\+Field$<$\+Type$>$ for cell-\/interior Gauss points
\item face\+Gauss\+Field$<$\+Type$>$ for Gauss points on cell faces
\end{DoxyItemize}

The class can be constructed\+:
\begin{DoxyItemize}
\item From a dof\+Field$<$\+Type$>$ to interpolate DOF values to Gauss points
\item As an empty container to be filled later
\item With a uniform initial value
\end{DoxyItemize}

\doxylink{classFoam_1_1GaussField}{Gauss\+Field} does not own mesh or DoF storage. All pointers are non-\/owning. It is intended to be stored per-\/cell inside higher-\/level DG field classes.
