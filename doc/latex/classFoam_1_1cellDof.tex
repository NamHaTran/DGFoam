Degree-\/of-\/freedom container for a DG cell. 


\begin{DoxyTemplParams}{Template Parameters}
{\em Type} & Field value type (e.\+g. scalar, vector, tensor)\\
\hline
\end{DoxyTemplParams}
\doxylink{classFoam_1_1cellDof}{cell\+Dof} stores all degrees of freedom (DoF) associated with a single DG cell for a given field type.

The Do\+Fs typically correspond to coefficients of polynomial basis functions defined on the reference cell and mapped to physical space.

This class is intentionally lightweight and contains no geometric information; it is purely a data container used by higher-\/level DG field and solver classes.
