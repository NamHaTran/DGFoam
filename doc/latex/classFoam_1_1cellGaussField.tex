Field values at Gauss points of a DG cell.

\doxylink{classFoam_1_1cellGaussField}{cell\+Gauss\+Field} stores field values evaluated at Gauss integration points inside a single DG cell.

The values are typically obtained by interpolating cell-\/local degrees of freedom (DoF) using basis functions provided by the DG geometric cell.

This class does not own geometric data or DoF storage; it holds non-\/owning pointers to \doxylink{classFoam_1_1dgGeomMesh}{dg\+Geom\+Mesh}, \doxylink{classFoam_1_1dgGeomCell}{dg\+Geom\+Cell}, and cell\+Dof$<$\+Type$>$.
