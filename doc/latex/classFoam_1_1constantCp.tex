\doxysubsection{Detailed Description}
Perfect-\/gas thermodynamic model with constant $C_p$. 

\doxylink{classFoam_1_1constantCp}{constant\+Cp} implements a thermodynamic model in which the specific heat at constant pressure is assumed to be constant.

The model reads\+: \begin{DoxyItemize}
\item $C_p$ from the {\ttfamily thermodynamics} sub-\/dictionary \item Molecular weight from the {\ttfamily specie} sub-\/dictionary\end{DoxyItemize}
The following thermodynamic properties are then derived\+: \begin{DoxyItemize}
\item Gas constant\+: $R = \frac{R_u}{W}$ \item Specific heat at constant volume\+: $C_v = C_p - R$ \item Heat-\/capacity ratio\+: $\gamma = \frac{C_p}{C_v}$\end{DoxyItemize}
Energy relations\+: \begin{DoxyItemize}
\item Internal energy\+: $e(T) = C_v T$ \item Enthalpy\+: $h(T) = C_p T$\end{DoxyItemize}
Speed of sound\+:  \[
  a = \sqrt{\gamma R T}
\]

\begin{DoxyParagraph}{Applicability}
\begin{DoxyItemize}
\item Ideal-\/gas equation of state \item Perfect-\/gas thermodynamics \item Compressible DG solvers\end{DoxyItemize}

\end{DoxyParagraph}
\begin{DoxyParagraph}{Limitations}
\begin{DoxyItemize}
\item Does not include formation enthalpy \item Not suitable for real-\/gas or reacting-\/flow thermodynamics \end{DoxyItemize}

\end{DoxyParagraph}
