\doxysubsection{Detailed Description}
Central (non-\/dissipative) numerical flux. 

Implements the classical central flux used in DG formulations.

The numerical flux is defined as the arithmetic average of the physical flux evaluated on the minus and plus sides\+:

\[
  \boldsymbol{h}
  =
  \frac{1}{2}
  \left(
    \boldsymbol{F}^{-} + \boldsymbol{F}^{+}
  \right)
\]

where\+:
\begin{DoxyItemize}
\item $\boldsymbol{F}^{-}$ is the flux from the interior (minus-\/side) state
\item $\boldsymbol{F}^{+}$ is the flux from the exterior (plus-\/side) state
\end{DoxyItemize}

No upwinding or numerical dissipation is applied. This flux is therefore non-\/dissipative and non-\/directional.

Typical use cases\+:
\begin{DoxyItemize}
\item Diffusive fluxes
\item LDG auxiliary equations
\item Elliptic operators with external stabilisation
\end{DoxyItemize}

This flux is generally unsuitable for convection-\/dominated hyperbolic problems without additional stabilisation.
