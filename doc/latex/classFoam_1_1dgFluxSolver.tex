\doxysubsection{Detailed Description}
Base class for DG numerical flux solvers. 

Defines the runtime-\/selectable interface for numerical-\/flux solvers that operate on \doxylink{classFoam_1_1faceGaussField}{face\+Gauss\+Field} data. Implementations compute a numerical flux from minus/plus states at face Gauss points.

Plus/minus convention\+:
\begin{DoxyItemize}
\item minus\+: owner cell (P-\/side)
\item plus \+: neighbour cell (N-\/side)
\end{DoxyItemize}

The equation\+Type is used by derived solvers to interpret the meaning of state components and to select wave-\/speed estimates where required.
