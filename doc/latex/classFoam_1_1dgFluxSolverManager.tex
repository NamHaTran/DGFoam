\doxysubsection{Detailed Description}
Manager for flux solvers selected per equation term. 

This class constructs and manages \doxylink{classFoam_1_1dgFluxSolver}{dg\+Flux\+Solver} instances based on the {\ttfamily flux\+Schemes} dictionary.

Each equation term (e.\+g. mass\+Conv\+Term\+Flux, momentum\+Conv\+Term\+Flux) can be mapped to a specific numerical flux solver. If a term is not explicitly defined, a default solver is used.

The manager supports\+:
\begin{DoxyItemize}
\item Term-\/specific flux solver selection
\item A global default solver fallback
\item Solver-\/specific coefficient sub-\/dictionaries
\end{DoxyItemize}

\begin{DoxyParagraph}{Example dictionary}
\begin{DoxyVerb}* fluxSchemes
* {
*     fluxSolvers
*     {
*         default                 central;
*
*         massConvTermFlux        HLLC;
*         momentumConvTermFlux    LaxFriedrichs;
*
*         // Other flux terms such as:
*         // - auxiliaryFlux
*         // - massDiffusiveFlux
*         // - momentumDiffusiveFlux
*         // - energyConvectiveFlux
*         // - energyDiffusiveFlux
*         // fall back to the default solver if not specified.
*     }
*
*     fluxSolversCoeffs
*     {
*         HLLCCoeffs
*         {
*             speedEstimate roeEinfeldt;
*         }
*
*         LaxFriedrichsCoeffs
*         {
*             scaleByMach true;
*         }
*
*         centralCoeffs
*         {
*             // no parameters
*         }
*     }
* }
* \end{DoxyVerb}

\end{DoxyParagraph}
Solver instances are created lazily using runtime selection tables and are reused across time steps. The \doxylink{classFoam_1_1dgFluxSolverManager_ae94120a1e88f153760d3fcdf661572bf}{reset()} method propagates to all managed solvers.
