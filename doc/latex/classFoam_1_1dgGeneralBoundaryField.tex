\doxysubsection{Detailed Description}
\subsubsection*{template$<$\mbox{\hyperlink{classFoam_1_1GaussField}{class}} \mbox{\hyperlink{classFoam_1_1GaussField}{Type}}$>$\newline
class Foam\+::dg\+General\+Boundary\+Field$<$ Type $>$}
Base class for DG ghost-\/state boundary conditions. 

Defines a runtime-\/selectable interface for DG boundary conditions that compute the exterior (plus-\/side) state from the interior (minus-\/side) state at boundary face Gauss points\+:

\[
  \phi^{-} \;\longrightarrow\; \phi^{+}
\]

Derived classes typically override\+:
\begin{DoxyItemize}
\item \doxylink{classFoam_1_1dgGeneralBoundaryField_aa7dbaa0dcb39c4f2e626d2becc772287}{update\+Ghost\+State\+Value()} to define the ghost-\/state value
\item \doxylink{classFoam_1_1dgGeneralBoundaryField_a7108337bba276840554a8e4caa6f526a}{update\+Ghost\+State\+Grad()} to define the ghost-\/state gradient
\end{DoxyItemize}

The default implementation corresponds to a copy/zero-\/gradient behaviour.
