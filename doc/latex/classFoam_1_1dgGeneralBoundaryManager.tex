\doxysubsection{Detailed Description}
\subsubsection*{template$<$\mbox{\hyperlink{classFoam_1_1GaussField}{class}} \mbox{\hyperlink{classFoam_1_1GaussField}{Type}}$>$\newline
class Foam\+::dg\+General\+Boundary\+Manager$<$ Type $>$}
Manager for DG general boundary conditions. 


\begin{DoxyTemplParams}{Template Parameters}
{\em Type} & Field value type (e.\+g. scalar, vector, tensor)\\
\hline
\end{DoxyTemplParams}
Constructs and stores one \doxylink{classFoam_1_1dgGeneralBoundaryField}{dg\+General\+Boundary\+Field} per fv\+Patch and applies them to update ghost (plus-\/side) values and gradients at face Gauss points.

The manager\+:
\begin{DoxyItemize}
\item Filters patches via \doxylink{namespaceFoam_1_1dgBoundaryHelper_ad336b0d03003430f2a70769a5a4ccaa3}{dg\+Boundary\+Helper\+::is\+General\+Patch()}
\item Builds boundary-\/condition objects via runtime selection
\item Dispatches update\+Ghost\+State\+Value()/update\+Ghost\+State\+Grad() per Gauss point 
\end{DoxyItemize}
