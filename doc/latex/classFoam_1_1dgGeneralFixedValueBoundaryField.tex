\doxysubsection{Detailed Description}
\subsubsection*{template$<$\mbox{\hyperlink{classFoam_1_1GaussField}{class}} \mbox{\hyperlink{classFoam_1_1GaussField}{Type}}$>$\newline
class Foam\+::dg\+General\+Fixed\+Value\+Boundary\+Field$<$ Type $>$}
Strong Dirichlet boundary condition for DG fields. 

Enforces a prescribed boundary value using a ghost-\/state reflection.

The Dirichlet condition is imposed strongly by mirroring the interior (minus-\/side) state about the prescribed value\+:

\[
  U^{+} = 2\,U_{\mathrm{bc}} - U^{-}
\]

where\+:
\begin{DoxyItemize}
\item $U^{-}$ is the interior (minus-\/side) value
\item $U^{+}$ is the ghost (plus-\/side) value
\item $U_{\mathrm{bc}}$ is the prescribed boundary value
\end{DoxyItemize}

No explicit gradient modification is applied; the boundary value fully constrains the solution at the interface.
