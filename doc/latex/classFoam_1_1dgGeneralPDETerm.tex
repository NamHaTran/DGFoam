\subsubsection*{template$<$\mbox{\hyperlink{classFoam_1_1GaussField}{class}} \mbox{\hyperlink{classFoam_1_1GaussField}{UType}}, \mbox{\hyperlink{classFoam_1_1GaussField}{class}} \mbox{\hyperlink{classFoam_1_1GaussField}{FType}}$>$\newline
class Foam\+::dg\+General\+PDETerm$<$ UType, FType $>$}
Generic DG PDE term for residual assembly. 


\begin{DoxyTemplParams}{Template Parameters}
{\em UType} & Type of solution variable (scalar or vector) \\
\hline
{\em FType} & Type of flux (vector or tensor)\\
\hline
\end{DoxyTemplParams}
This class represents a single PDE operator contribution to the DG residual of one cell.

The residual is assembled in the form\+:  \[
R_i
=
- \int_{\Omega}
    \nabla \phi_i : \boldsymbol{F}\,\mathrm{d}V
+ \int_{\partial\Omega}
    \phi_i (\boldsymbol{F}^{\ast}\cdot\boldsymbol{n})\,\mathrm{d}S
\]

Supported type combinations\+:
\begin{DoxyItemize}
\item UType = scalar, FType = vector
\item UType = vector, FType = tensor
\end{DoxyItemize}

The numerical surface flux $\boldsymbol{F}^{\ast}$ is computed through \doxylink{classFoam_1_1dgFluxSolverManager}{dg\+Flux\+Solver\+Manager} based on the PDE term name.

This class\+:
\begin{DoxyItemize}
\item Operates on a single cell
\item Owns no global data
\item Is used repeatedly during residual assembly 
\end{DoxyItemize}
