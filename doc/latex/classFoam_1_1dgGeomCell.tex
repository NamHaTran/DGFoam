\doxysubsection{Detailed Description}
Geometric container for a DG cell. 

The \doxylink{classFoam_1_1dgGeomCell}{dg\+Geom\+Cell} class represents the geometry of a single mesh cell tailored for Discontinuous Galerkin discretization.

It provides\+:
\begin{DoxyItemize}
\item Access to mesh-\/based quantities (volume, centroid)
\item Cell topology (faces, neighbors, vertices)
\item Access to reference-\/cell data (Gauss points, weights, basis functions)
\end{DoxyItemize}

Geometry-\/specific face objects (\doxylink{classFoam_1_1dgGeomFace}{dg\+Geom\+Face}) are initialized using a shared reference face (\doxylink{classFoam_1_1dgRefFace}{dg\+Ref\+Face}). Reference-\/cell data is shared via \doxylink{classFoam_1_1dgRefCell}{dg\+Ref\+Cell} to avoid duplication and ensure consistency.
