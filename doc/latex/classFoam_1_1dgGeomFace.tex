\doxysubsection{Detailed Description}
Geometric face data for DG discretization. 

\doxylink{classFoam_1_1dgGeomFace}{dg\+Geom\+Face} represents a single mesh face in the DG framework. It provides\+:
\begin{DoxyItemize}
\item Access to geometric quantities (area, normal, centre)
\item Face orientation relative to owner and neighbour cells
\item Mapping between reference and physical face coordinates
\item Gauss-\/point connectivity and basis-\/function data for surface integrals
\end{DoxyItemize}

The face uses shared reference-\/face data (\doxylink{classFoam_1_1dgRefFace}{dg\+Ref\+Face}) to evaluate basis functions and quadrature rules consistently across all faces of the same type.
