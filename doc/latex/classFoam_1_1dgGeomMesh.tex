\doxysubsection{Detailed Description}
DG geometric mesh wrapper around fv\+Mesh. 

\doxylink{classFoam_1_1dgGeomMesh}{dg\+Geom\+Mesh} constructs and stores all DG-\/specific geometric entities derived from an fv\+Mesh.

Responsibilities include\+:
\begin{DoxyItemize}
\item Creating and storing \doxylink{classFoam_1_1dgGeomFace}{dg\+Geom\+Face} and \doxylink{classFoam_1_1dgGeomCell}{dg\+Geom\+Cell} objects
\item Managing shared reference-\/face (\doxylink{classFoam_1_1dgRefFace}{dg\+Ref\+Face}) data
\item Managing reference-\/cell (\doxylink{classFoam_1_1dgRefCell}{dg\+Ref\+Cell}) data for different cell types
\item Providing connectivity information required for DG surface and volume integration
\end{DoxyItemize}

The class acts as the central geometric data container for DG discretization.
