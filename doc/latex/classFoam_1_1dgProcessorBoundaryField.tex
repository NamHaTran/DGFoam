\subsubsection*{template$<$\mbox{\hyperlink{classFoam_1_1GaussField}{class}} \mbox{\hyperlink{classFoam_1_1GaussField}{Type}}$>$\newline
class Foam\+::dg\+Processor\+Boundary\+Field$<$ Type $>$}
DG processor boundary handler for parallel communication. 


\begin{DoxyTemplParams}{Template Parameters}
{\em Type} & Field value type (e.\+g. scalar, vector, tensor)\\
\hline
\end{DoxyTemplParams}
Exchanges DG face Gauss-\/point plus-\/side values across processor patches.

The communication uses explicit send/receive buffers and point-\/to-\/point transfers on processor\+Fv\+Patch.

MPI tags are generated from processor ranks to avoid collisions.
