\doxysubsection{Detailed Description}
\subsubsection*{template$<$\mbox{\hyperlink{classFoam_1_1GaussField}{class}} \mbox{\hyperlink{classFoam_1_1GaussField}{Type}}$>$\newline
class Foam\+::dg\+Processor\+Boundary\+Field$<$ Type $>$}
DG processor boundary handler for parallel communication. 


\begin{DoxyTemplParams}{Template Parameters}
{\em Type} & Field value type (e.\+g. scalar, vector, tensor)\\
\hline
\end{DoxyTemplParams}
Exchanges DG face Gauss-\/point interface states across processor patches.

Each side sends its local minus-\/side trace on the processor patch and stores the received neighbour trace as the local plus-\/side value.

When the DG field owns Do\+Fs, the communication also transfers the owner-\/cell modal coefficients for each processor face so face-\/local neighbour DoF buffers can be populated for downstream algorithms such as DG limiters.

The communication uses explicit send/receive buffers and Open\+FOAM non-\/blocking processor streams on processor\+Fv\+Patch.
