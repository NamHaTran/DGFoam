\subsubsection*{template$<$\mbox{\hyperlink{classFoam_1_1GaussField}{class}} \mbox{\hyperlink{classFoam_1_1GaussField}{Type}}$>$\newline
class Foam\+::dg\+Processor\+Boundary\+Manager$<$ Type $>$}
Manager for DG processor boundary fields. 


\begin{DoxyTemplParams}{Template Parameters}
{\em Type} & Field value type (e.\+g. scalar, vector, tensor)\\
\hline
\end{DoxyTemplParams}
Detects processor patches in the fv\+Mesh and constructs one \doxylink{classFoam_1_1dgProcessorBoundaryField}{dg\+Processor\+Boundary\+Field} per processor patch.

The manager coordinates MPI exchanges to synchronise plus-\/side Gauss values on processor faces. Physical boundary conditions are handled separately by general boundary managers.
