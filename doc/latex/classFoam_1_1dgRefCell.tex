\doxysubsection{Detailed Description}
Reference-\/cell data for DG discretization. 

\doxylink{classFoam_1_1dgRefCell}{dg\+Ref\+Cell} stores all reference-\/space data required by the Discontinuous Galerkin method for a given cell type and polynomial order.

This includes\+:
\begin{DoxyItemize}
\item Gauss quadrature points and weights in reference coordinates
\item Basis functions evaluated at Gauss points
\item Derivatives of basis functions with respect to reference coordinates $\eta_1$, $\eta_2$, and $\eta_3$
\end{DoxyItemize}

The data is generated once and shared among all physical cells of the same type and order to avoid redundant computations.
