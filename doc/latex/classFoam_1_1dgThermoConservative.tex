\doxysubsection{Detailed Description}
Thermodynamic manager for DG solvers using conservative variables. 

\doxylink{classFoam_1_1dgThermoConservative}{dg\+Thermo\+Conservative} reconstructs all primitive thermodynamic quantities from the conserved variables $(\rho, \rho\mathbf{U}, E)$ at each Gauss point of every DG cell.

The class provides temperature, pressure, internal energy, enthalpy, heat capacities, heat-\/capacity ratio, speed of sound, and transport properties in a fully cell-\/local and Gauss-\/point-\/resolved manner.

\begin{DoxyParagraph}{Design intent}

\begin{DoxyItemize}
\item Designed for compressible DG solvers using conservative formulations.
\item All thermodynamic reconstruction is performed locally at Gauss points.
\item Acts as a wrapper coordinating EOS, \doxylink{classFoam_1_1thermoLaw}{thermo\+Law}, \doxylink{classFoam_1_1transportLaw}{transport\+Law}, and energy models.
\end{DoxyItemize}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Supported model families}

\begin{DoxyItemize}
\item Ideal-\/gas and thermal-\/perfect-\/gas EOS
\item Compatible \doxylink{classFoam_1_1thermoLaw}{thermo\+Law} and \doxylink{classFoam_1_1transportLaw}{transport\+Law} models
\item Internal-\/energyâ€“based formulations (DG conservative)
\end{DoxyItemize}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Notes}

\begin{DoxyItemize}
\item Primitive-\/variable formulations are not supported.
\item Incompressible models are not supported.
\item DG conservative formulation uses internal energy only. 
\end{DoxyItemize}
\end{DoxyParagraph}
