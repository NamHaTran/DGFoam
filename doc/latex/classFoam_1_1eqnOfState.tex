\doxysubsection{Detailed Description}
Base class for single-\/phase equations of state. 

This class defines the interface for single-\/phase equations of state (EOS) used in DGFoam. An EOS provides thermodynamic closure relations between pressure, temperature, and density, and supplies classification flags that allow thermo models to verify physical compatibility.

\begin{DoxyParagraph}{EOS classification and compatibility}
To ensure consistency between EOS models and \doxylink{classFoam_1_1thermoLaw}{thermo\+Law} formulations, each EOS advertises a set of capability and classification flags. Higher-\/level thermo models must check these flags in their constructors to ensure that the selected EOS is physically compatible with their assumptions.
\end{DoxyParagraph}
EOS models are grouped into the following categories\+:

\begin{DoxyParagraph}{Ideal-\/gas–type EOS}

\begin{DoxyItemize}
\item Identified by \doxylink{classFoam_1_1eqnOfState_a0d4f0890fe5145ecaa8bdb16897773e4}{is\+Ideal\+Gas()} returning true.
\item Pressure relation of the form\+: $ p = \rho R T $.
\item Includes classic ideal gas and thermal-\/perfect gas variants.
\item Thermodynamic properties (Cp, Cv, e, h) are provided by \doxylink{classFoam_1_1thermoLaw}{thermo\+Law}.
\item Compatible with most \doxylink{classFoam_1_1thermoLaw}{thermo\+Law} models.
\end{DoxyItemize}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Incompressible EOS}

\begin{DoxyItemize}
\item Identified by \doxylink{classFoam_1_1eqnOfState_ae9c9acc43e32e77440222dc2b2d3ec8a}{is\+Incompressible()} returning true.
\item Density is constant or independent of pressure and temperature.
\item Pressure does not participate in thermodynamic relations.
\item Only compatible with thermo models designed for liquids.
\end{DoxyItemize}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Real-\/gas EOS}

\begin{DoxyItemize}
\item Identified by \doxylink{classFoam_1_1eqnOfState_a4e8808415e2f46a0e000e9d5337565f2}{is\+Real\+Gas()} returning true.
\item Non-\/ideal equations of state (e.\+g. Peng–\+Robinson, Redlich–\+Kwong, stiffened gas, Van der Waals).
\item Require thermo models derived for real-\/gas properties.
\end{DoxyItemize}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Kinetic-\/theory EOS}

\begin{DoxyItemize}
\item Identified by \doxylink{classFoam_1_1eqnOfState_a5cdac2918c31d6362f277a19dba79d08}{is\+Thermal\+Perfect\+Gas()} returning true.
\item EOS derived from kinetic theory (Boltzmann / ES-\/\+BGK).
\item Pressure may originate from stress rather than $ p = \rho R T $.
\item Used for rarefied or high-\/\+Knudsen-\/number flows.
\end{DoxyItemize}
\end{DoxyParagraph}
By default, all classification flags return false in this base class. Derived EOS models override the appropriate flags to identify their physical category.
