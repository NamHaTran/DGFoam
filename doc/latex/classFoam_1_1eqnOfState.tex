\doxysubsection{Detailed Description}
Base class for single-\/phase equations of state. 

This class defines the interface for single-\/phase equations of state (EOS) used in DGFoam. An EOS provides thermodynamic closure relations between pressure, temperature, and density, and supplies classification flags that allow thermo models to verify physical compatibility.

\begin{DoxyParagraph}{EOS classification and compatibility}
To ensure consistency between EOS models and \doxylink{classFoam_1_1thermoLaw}{thermo\+Law} formulations, each EOS advertises a set of capability and classification flags. Higher-\/level thermo models must check these flags in their constructors to ensure that the selected EOS is physically compatible with their assumptions.
\end{DoxyParagraph}
EOS models are grouped into the following categories\+:

\begin{DoxyParagraph}{Ideal-\/gas–type EOS}

\begin{DoxyItemize}
\item Identified by \doxylink{classFoam_1_1eqnOfState_a0d4f0890fe5145ecaa8bdb16897773e4}{is\+Ideal\+Gas()} returning true.
\item Pressure relation of the form\+: $ p = \rho R T $.
\item Includes classic ideal gas and thermal-\/perfect gas variants.
\item Thermodynamic properties (Cp, Cv, e, h) are provided by \doxylink{classFoam_1_1thermoLaw}{thermo\+Law}.
