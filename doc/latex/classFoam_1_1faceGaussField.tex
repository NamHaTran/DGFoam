class Foam\+::face\+Gauss\+Field$<$ Type $>$}
Gauss-\/point field on faces of a DG cell. 


\begin{DoxyTemplParams}{Template Parameters}
{\em Type} & Field value type (e.\+g. scalar, vector, tensor)\\
\hline
\end{DoxyTemplParams}
\doxylink{classFoam_1_1faceGaussField}{face\+Gauss\+Field} stores field values evaluated at Gauss integration points on all faces of a DG cell.

For each Gauss point on a face, the class stores\+:
\begin{DoxyItemize}
\item Minus-\/side value (owner cell)
\item Plus-\/side value (neighbour cell or boundary)
\end{DoxyItemize}

The class is designed for cell-\/centric DG formulations and supports flux computation across cell interfaces.

This class does not own geometry or DoF storage; it holds non-\/owning pointers to \doxylink{classFoam_1_1dgGeomMesh}{dg\+Geom\+Mesh}, \doxylink{classFoam_1_1dgGeomCell}{dg\+Geom\+Cell}, \doxylink{classFoam_1_1dgGeomFace}{dg\+Geom\+Face}, and cell\+Dof$<$\+Type$>$.
