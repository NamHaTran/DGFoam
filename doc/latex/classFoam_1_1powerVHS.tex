\doxysubsection{Detailed Description}
Variable-\/\+Hard-\/\+Sphere (VHS) power-\/law viscosity model. 

\doxylink{classFoam_1_1powerVHS}{power\+VHS} implements the Variable-\/\+Hard-\/\+Sphere (VHS) model with a power-\/law temperature dependence of viscosity\+:

\[
  \mu(T) = \mu_{\mathrm{ref}}
           \left(\frac{T}{T_{\mathrm{ref}}}\right)^{\omega}
\]

The reference viscosity is computed from kinetic theory as\+:

\[
  \mu_{\mathrm{ref}} =
  \frac{15 \sqrt{\pi m k_B T_{\mathrm{ref}}}}
       {2 \pi d_{\mathrm{ref}}^2
        (5 - 2\omega)(7 - 2\omega)}
\]

where\+: \begin{DoxyItemize}
\item $m$ is the molecular mass \item $k_B$ is the Boltzmann constant \item $d_{\mathrm{ref}}$ is the reference molecular diameter \item $T_{\mathrm{ref}}$ is the reference temperature \item $\omega$ is the VHS power-\/law exponent\end{DoxyItemize}
The Prandtl number is assumed constant\+:  \[
  \mathrm{Pr}(T) = \mathrm{Pr}_0
\]

Thermal conductivity is computed in the base \doxylink{classFoam_1_1transportLaw}{transport\+Law} as\+:  \[
  \kappa(T) = \frac{\mu(T)\,C_p(T)}{\mathrm{Pr}(T)}
\]

\begin{DoxyParagraph}{Notes}
\begin{DoxyItemize}
\item Molecular mass is read in $\mathrm{g/mol}$ and converted internally to $\mathrm{kg/molecule}$ \item $d_{\mathrm{ref}}$ is specified in $\mathrm{m}$ \item $T_{\mathrm{ref}}$ is specified in $\mathrm{K}$ \item $k_B$ is specified in $\mathrm{J/K}$ \item $N_A$ is specified in $\mathrm{mol^{-1}}$\end{DoxyItemize}

\end{DoxyParagraph}
