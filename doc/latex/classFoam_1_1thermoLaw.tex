\doxysubsection{Detailed Description}
Base class for thermodynamic laws in DG solvers. 

\doxylink{classFoam_1_1thermoLaw}{thermo\+Law} defines the thermodynamic closure relations required by DG thermophysical models. It provides a unified interface for\+:

\begin{DoxyItemize}
\item Converting between temperature and energy forms $(e, h)$ \item Computing heat capacities $C_p(T)$, $C_v(T)$ \item Computing heat-\/capacity ratio $\gamma(T)$ \item Computing speed of sound $a(T)$ \item Evaluating internal energy and enthalpy at Gauss points\end{DoxyItemize}
All operations are performed locally at Gauss points of each DG cell, consistent with high-\/order discontinuous Galerkin formulations.

\end{DoxyParagraph}
\begin{DoxyParagraph}{Perfect-\/gas thermodynamics}
\begin{DoxyItemize}
\item $C_p(T)$ and $C_v(T)$ depend only on temperature \item No formation enthalpy is included \item $h(T) = \int C_p(T)\,\mathrm{d}T$ \item $e(T) = \int C_v(T)\,\mathrm{d}T$ \item Compatible with ideal-\/gas EOS\+: $p = \rho R T$ \item Used in compressible and high-\/temperature gas flows\end{DoxyItemize}

\end{DoxyParagraph}
\begin{DoxyParagraph}{Absolute-\/enthalpy thermodynamics}
\begin{DoxyItemize}
\item Enthalpy includes chemical formation energy \item $h(T) = h_f^0 + \int C_p(T)\,\mathrm{d}T$ \item Required for combustion and reacting flows \item Works with ideal-\/gas or selected real-\/gas EOS\end{DoxyItemize}

\end{DoxyParagraph}
\begin{DoxyParagraph}{Real-\/gas thermodynamics}
\begin{DoxyItemize}
\item $C_p$, $C_v$, $e$, and $h$ depend on both temperature and density \item Must be used together with real-\/gas EOS \item Required for high-\/pressure and supercritical fluids\end{DoxyItemize}
